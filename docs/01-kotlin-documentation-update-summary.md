**파일 경로**: `docs/01-1-kotlin-history.md`

**목적**: Kotlin의 탄생 배경, 역사, 철학을 상세히 설명

**주요 섹션**:

1. **Kotlin의 탄생**
   - JetBrains의 고민과 문제 발견
   - Java의 한계점 분석
   - 대안 검토 (Scala, Groovy)
   - 새로운 언어 개발 결정

2. **왜 Kotlin이 만들어졌나?**
   - JetBrains의 5가지 목표
   - 비즈니스 동기
   - 실용주의 철학

3. **Kotlin의 발전 과정**
   - 2010년: 프로젝트 시작
   - 2011년: 공개 발표
   - 2012년: 오픈소스화
   - 2016년: Kotlin 1.0 출시
   - 2017년: Google의 Android 공식 언어 채택 🚀
   - 2019년: Kotlin First 선언
   - 2021년: Jetpack Compose 출시
   - 2024년: Kotlin 2.0 출시

4. **Android와 Kotlin**
   - Google이 Kotlin을 선택한 이유
   - Android에서의 성장 (채택률 80%+)
   - Jetpack Compose의 등장

5. **Kotlin의 철학**
   - 5가지 핵심 원칙 (실용성, 간결성, 안전성, 상호운용성, 도구 친화성)
   - 설계 철학
   - 점진적 학습 곡선

6. **Kotlin의 영향력**
   - Kotlin Multiplatform (KMP)
   - 교육과 학습
   - 기업 채택 현황
   - 통계와 인기도
   - 미래 전망
