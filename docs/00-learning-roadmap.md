## 💡 학습 팁

### 효과적인 학습 방법

1. **역사와 철학 먼저 이해하기**
   - "왜" 이 기술이 나왔는지 이해하면 학습이 훨씬 쉬워집니다.
   - 각 주제의 `-1` 문서를 먼저 읽으세요.

2. **이론 → 실습 → 프로젝트**
   - 문서를 읽고 (30%)
   - 예제를 직접 타이핑해보고 (40%)
   - 작은 프로젝트를 만들어본다 (30%)

3. **매일 조금씩**
   - 하루 1-2시간씩 꾸준히
   - 주말에 프로젝트 시간 확보

4. **커뮤니티 활용**
   - Stack Overflow
   - Reddit r/androiddev
   - Kotlin Slack
