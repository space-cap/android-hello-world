# 심플 뽀모도로 타이머 (Focus Flow)

**날짜:** 2025-12-09
**상태:** 초안 (Draft)
**작성자:** [사용자 이름]

## 1. 개요 (Summary)
가장 클래식한 시간 관리 기법인 '뽀모도로(Pomodoro)'를 아름다운 시각 효과와 함께 제공하는 타이머 앱입니다. 25분 집중, 5분 휴식의 리듬을 놓치지 않도록 돕습니다.

## 2. 배경 및 문제 의식 (Background & Problem)
*   **산만한 환경:** 스마트폰 알림 등으로 인해 깊이 집중하기 어려운 환경입니다.
*   **기존 타이머의 한계:** 일반 타이머는 남은 시간만 보여주어 집중/휴식 사이클 관리가 불편합니다.
*   **시각적 피드백 부재:** 시간이 흐르는 것을 숫자로만 보면 압박감을 느끼기 쉽습니다.

## 3. 제안 내용 (Proposal)

### 3.1 핵심 컨셉
*   **Visual Progress:** 화면 전체에 물이 차오르거나, 원형 그래프가 줄어드는 등 '흐름'을 시각화합니다.
*   **Distraction Free:** 타이머 작동 중에는 다른 앱 알림을 최소화하거나 방해 금지 모드를 제안합니다. (기술적 제약 내에서)

### 3.2 주요 기능
1.  **타이머 사이클 (Cycle Manager)**
    *   **집중 (Focus):** 기본 25분. 붉은색 테마.
    *   **휴식 (Short Break):** 기본 5분. 초록색/푸른색 테마.
    *   **긴 휴식 (Long Break):** 4 사이클 후 15~30분 휴식 제안.
    *   커스텀: 사용자 집중력에 맞춰 시간 설정 가능 (예: 50분/10분).

2.  **화이트 노이즈 (Background Noise)**
    *   집중을 돕는 ASMR 사운드(빗소리, 장작 타는 소리, 카페 소음) 재생 기능 탑재.

3.  **백그라운드 실행 (Foreground Service)**
    *   앱을 내려도 타이머는 계속 작동하며, 알림창(Notification)에서 남은 시간 확인 및 일시정지 가능.

### 3.3 UI/UX 스케치
*   **메인 화면:** 중앙에 거대한 원형 프로그레스 바. 타이머 시작 시 부드러운 애니메이션과 함께 시간이 줄어듦.
*   **전환 효과:** 집중 -> 휴식 모드 전환 시 배경색이 부드럽게 바뀜 (Morphing Color).

## 4. 기대 효과 (Benefits)
*   **생산성 증대:** 짧고 굵게 집중하는 습관을 형성하여 업무/학습 효율 향상.
*   **번아웃 방지:** 강제적인 휴식 시간을 통해 뇌의 피로도 관리.
*   **심미적 만족:** 책상 위에 켜두기만 해도 인테리어 소품 같은 느낌을 주는 Digital Desk Toy.

## 5. 기술적 고려사항 (Technical Considerations)
*   **Foreground Service:**
    *   앱이 백그라운드로 가거나 화면이 꺼져도 타이머가 정확히 동작해야 함.
    *   Notification에 진행 상황(Progress Bar) 표시.
*   **Canvas API (Jetpack Compose):**
    *   원형 프로그레스 바, 물결 효과 등을 직접 그리기 위해 Canvas 활용.
    *   부드러운 프레임 처리를 위한 최적화 필요.
*   **Audio Focus:**
    *   화이트 노이즈 재생 시 다른 음악 앱과의 오디오 포커스 처리 (음악과 섞어 들을지, 멈출지 등).
*   **Vibration:** 타이머 종료 시 햅틱 피드백 패턴 구현.
