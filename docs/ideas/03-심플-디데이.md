# 심플 디데이 (Simple D-Day: Focus & Glance)

**날짜:** 2025-12-09
**상태:** 초안 (Draft)
**작성자:** [사용자 이름]

## 1. 개요 (Summary)
복잡한 기능 없이 오직 '남은 시간' 확인이라는 본질에 집중한 미니멀리즘 디데이 앱입니다. 홈 화면 위젯을 통해 앱을 열지 않고도 중요한 일정을 직관적으로 확인할 수 있습니다.

## 2. 배경 및 문제 의식 (Background & Problem)
*   **과도한 기능:** 기존 디데이 앱들은 커뮤니티, 쇼핑 등 불필요한 기능이 많아 무겁습니다.
*   **시인성 부족:** 화려한 배경 이미지 때문에 정작 중요한 '남은 날짜' 텍스트가 잘 보이지 않는 경우가 많습니다.
*   **위젯의 중요성:** 사용자는 앱 진입보다 홈 화면에서 바로 확인하기를 원합니다.

## 3. 제안 내용 (Proposal)

### 3.1 핵심 컨셉
*   **Glanceable:** 0.1초 만에 정보 파악 가능.
*   **Typographic Design:** 이미지보다는 아름다운 폰트와 타이포그래피로 감성 전달.
*   **Widget First:** 앱 기능의 50% 이상을 위젯 경험에 투자.

### 3.2 주요 기능
1.  **D-Day 리스트 (Main List)**
    *   **임박순 정렬:** 가장 가까운 일정이 최상단에 카드 형태로 강조되어 표시.
    *   **다양한 카운트:** D-Day (시험 등), D+Day (연인, 금연 등), 반복 일정 (매년 생일).

2.  **초간단 등록 (Quick Add)**
    *   제목과 날짜만 입력하면 끝.
    *   배경색상이나 이모지 아이콘으로 간단히 구분 (이미지 업로드 X).

3.  **홈 화면 위젯 (App Widget)**
    *   **1x1:** 가장 중요한 하나의 D-Day 만 심플하게 표시.
    *   **2x1, 4x1:** 임박한 일정 3개를 리스트 형태로 표시.
    *   **디자인 커스텀:** 위젯 배경 투명도, 텍스트 색상 조절.

### 3.3 UI/UX 스케치
*   **메인 화면:** 여백이 많은 화이트/다크 모드 디자인.
*   **카드 디자인:** 큼지막한 숫자(D-10)가 시선을 사로잡음. 부가 정보는 작게 표시.

## 4. 기대 효과 (Benefits)
*   **집중력 향상:** 수능, 자격증 시험 등 목표일까지 남은 시간을 시각화하여 동기 부여.
*   **편의성:** 위젯 지원으로 접근성을 극대화하여 사용자 리텐션 유지.
*   **개발 효율:** 복잡한 서버 로직 없이 로컬 기능만으로 완성도 높은 앱 제작 가능.

## 5. 기술적 고려사항 (Technical Considerations)
*   **Local Storage:**
    *   **DataStore (Proto):** 가벼운 데이터 구조라 DB 대신 DataStore 사용 고려 가능, 혹은 확장성을 위해 Room 사용. (이 프로젝트에선 일관성을 위해 Room 추천)
*   **Jetpack Glance:** 최신 Compose 스타일로 앱 위젯 개발. (기존 XML 방식보다 생산성 높음)
*   **Date Logic:** `java.time.LocalDate`, `ChronoUnit` 을 사용하여 정확한 DD-Day 계산 (윤년 등 처리).
*   **WorkManager:** 자정이 지났을 때 위젯의 날짜를 갱신하기 위한 백그라운드 작업.
