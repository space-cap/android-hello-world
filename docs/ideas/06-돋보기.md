# 돋보기 (Smart Magnifier)

**날짜:** 2025-12-09
**상태:** 초안 (Draft)
**작성자:** [사용자 이름]

## 1. 개요 (Summary)
작은 글씨가 안 보여 불편한 순간, 스마트폰을 완벽한 디지털 돋보기로 변신시켜 주는 유틸리티 앱입니다. 단순 확대 기능을 넘어 '정지 화면(Freeze)'과 '고대비 필터'를 통해 시각적 편의를 극대화합니다.

## 2. 배경 및 문제 의식 (Background & Problem)
*   **노안 및 저시력:** 중장년층뿐만 아니라 시력이 좋지 않은 사람들은 약병 표기, 식당 메뉴판 등 작은 글씨를 읽는 데 어려움을 겪습니다.
*   **손떨림:** 글씨를 읽으려고 폰을 들고 있으면 손떨림 때문에 초점이 잘 맞지 않아 어지러움을 유발합니다.
*   **어두운 환경:** 식당이나 침실 등 조명이 어두운 곳에서는 글씨가 더 안 보입니다.

## 3. 제안 내용 (Proposal)

### 3.1 핵심 컨셉
*   **One-Hand Operation:** 줌 조절, 조명, 정지 등 모든 기능을 한 손가락으로 제어 가능하도록 UI 설계.
*   **Crystal Clear:** CameraX의 고해상도 프리뷰와 이미지 처리 기술을 활용해 선명한 화질 제공.

### 3.2 주요 기능
1.  **실시간 확대/축소 (Zoom)**
    *   슬라이더(SeekBar) 또는 핀치 줌(Pinch-to-Zoom)으로 부드러운 확대 조절.
    *   CameraX `CameraControl.setZoomRatio` 활용.

2.  **정지 화면 (Freeze Frame)** ⭐ 핵심 기능
    *   화면을 탭 하거나 버튼을 눌러 현재 화면을 일시 정지(캡처).
    *   **이점:** 손을 떨지 않고 편안하게 확대해서 글씨를 읽을 수 있음. 원본 화질 유지하며 Pan & Zoom 지원.

3.  **필터 모드 (High Contrast)**
    *   **반전(Inverse):** 흰 배경/검은 글씨 -> 검은 배경/흰 글씨로 반전 (눈의 피로 감소).
    *   **흑백(Grayscale):** 색맹/색약 사용자를 위한 대비 강조.

4.  **손전등 (Flashlight)**
    *   어두운 곳에서 바로 켤 수 있는 토치(Torch) 기능.

### 3.3 UI/UX 스케치
*   **Full Screen Preview:** 화면 전체를 꽉 채운 뷰파인더.
*   **Floating Controls:** 화면 하단에 반투명한 컨트롤 패널 배치 (줌 슬라이더, 셔터/프리즈 버튼 등).

## 4. 기대 효과 (Benefits)
*   **접근성 개선:** 시력 약자에게 필수적인 도구가 되어 디지털 접근성 향상에 기여.
*   **실생활 유용성:** 부모님 효도 앱 1순위. 설치해 드리면 매일 쓰시는 앱.
*   **기술 습득:** 안드로이드 카메라 API와 이미지 처리 기초를 탄탄하게 다질 수 있음.

## 5. 기술적 고려사항 (Technical Considerations)
*   **CameraX:**
    *   `PreviewView`: 카메라 화면 출력.
    *   `ImageAnalysis`: 실시간 프레임 분석 (고대비 필터 적용 시 필요할 수 있음).
    *   `Focus & Metering`: 터치한 영역에 초점 맞추기.
*   **Image Processing:**
    *   `Bitmap`: 정지 화면 캡처 시 비트맵 생성 및 메모리 관리.
    *   `ColorMatrixColorFilter`: Compose 또는 View 시스템에서 색상 반전 필터 적용.
*   **Gesture:** `TransformableState` (Compose)를 이용한 이미지 확대/이동(Pan/Zoom) 구현.
